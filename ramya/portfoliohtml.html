Highlighted Projects

Blood Cell Segmentation

A blood cell segmentation project aims to automate the analysis of microscopic blood smear images for medical diagnosis. The process begins
with dataset preparation, using annotated datasets like the Blood Cell Count Dataset or ALL-IDB. Images undergo preprocessing techniques,
such as contrast enhancement and noise reduction, to standardize inputs. Advanced models like U-Net, Mask R-CNN, or fully convolutional networks (FCNs)
are employed for pixel-level segmentation to identify and isolate red blood cells (RBCs), white blood cells (WBCs), and platelets.

Technology Stack:

  • Programming Languages: Python, Matlab
  • IDE: Google Colab
  • Database: MySQL

Pothole Detection System

This project develops an automated Pothole Detection System using Convolutional Neural Networks (CNNs) to identify potholes

Technology Stack:

  • Programming Languages: Python, Matlab
  • IDE: Google Colab, Visual Studio Code
  • Database: MySQL
